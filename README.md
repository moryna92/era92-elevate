## 🚀 Deploy to Vercel (5 minutes, free)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up if you don't have one.

### Step 2 — Create a new repository
1. Click the **+** button (top right) → **New repository**
2. Name it: `era92-superstriker`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. Click **uploading an existing file** on the repo page
2. Drag and drop the entire contents of this folder (all files and folders)
3. Click **Commit changes**

### Step 4 — Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click **Add New Project**
3. Select your `era92-superstriker` repository
4. Click **Deploy** — Vercel handles everything automatically

### Step 5 — Share with the team
Vercel gives you a live URL like:
`https://era92-superstriker.vercel.app`

Share that link with all 12 team members. Works on phone and desktop. ✅
